# CRL Java Pdfwriter
A simple library to generate Pdf stream Implemented with pure java.

## Features
- generate Pdf stream
- Create pages in random order
- Insert text (with font, color, transparency, rotation)
- Insert images (jpeg/png)
- Insert metadata
- draw line
- draw retangle
