# CRL Java PDFWriter
A simple library to generate Pdf stream implemented in pure java.

## Features
- generate Pdf stream
- create pages in random order
- insert text (with font, color, transparency, rotation)
- insert images (jpeg/png)
- insert metadata
- draw line
- draw retangle
